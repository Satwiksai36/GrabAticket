# District Hub Tickets

## Project Overview

District Hub Tickets is a comprehensive ticket booking platform designed for district-level events and services.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## getting Started

To get started with development:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Scripts

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run preview`: Preview build
- `npm run lint`: Lint code
